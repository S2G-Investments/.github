# S2G-Investments/.github

This repository holds S2G Investments' **public** GitHub organization profile —
the page shown at [github.com/S2G-Investments](https://github.com/S2G-Investments).

The public profile (`profile/README.md`) is intentionally high-level and contains
no internal URLs, project identifiers, service names, or personnel details.

The detailed engineering services directory — internal app URLs, repository
inventory, owners, and last-commit data — is maintained in a **private**,
members-only repository (`.github-private`) and is not exposed publicly.
